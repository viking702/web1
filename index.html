HyperText Markup Language (HTML, Cultural Language: Super Text Markup Language, Hyper Text Markup Language) is the dominant markup language for web pages. HTML allows you to create structured documents that provide links, citations and other items as well as expand the structural meaning for text such as headings, paragraphs, list submissions. And it can embed images and creation and create interactive forms. HTML is written in the form of HTML elements with "tags" in angle brackets in the web page content.
